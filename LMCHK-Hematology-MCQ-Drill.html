使用说明 How this works

这是什么？

2003–2022 年 LMCHK 内科考试中 168 道血液 / 免疫 / 肿瘤真题的离线题库。严格遵循 lmchk-clinical-study 技能的学习引擎规则。

为什么没有"标准答案"？

Adrian Lui 的 PDF 编译只含题干、不含答案。技能铁律 #2 是"不可编造医学事实" — 所以这里不会塞一份我捏造的答案表来制造虚假安全感。你是血液专科医生，你自己判对错；不确定时用每题底部的「📋 11 步分析 prompt」按钮，把它丢给 Claude 走完整解析框架。

Leitner 5 盒怎么走？

  • Box 0 新题（未做过）
  • Box 1 → 2 天后到期
  • Box 2 → 5 天后到期
  • Box 3 → 10 天后到期
  • Box 4 → 21 天后到期
  • Box 5 毕业 ✓
  • 答对 → 升盒，答错 → 跌回 Box 1
  • "跨多个日期连续答对才毕业"——不掌握不放过

每题的流程

  • 选 A–E（也可用键盘 1/2/3/4/5）
  • 报信心 ① 蒙的 / ② 不太确定 / ③ 很确定
  • 自评 ✓ 对 / ✗ 错（键盘 Y / N）
  • 答错时标错因（知识缺失 / 概念混淆 / 题干误读 / 推理断链 / 粗心）
  • 信心 ③ 但答错 = 🔥 超校正黄金时刻——账本最高优先级
  • 笔记区记录官方答案、单词收割、临床珍珠

键盘快捷键

诚实承认边界

本工具擅长：单题深挖、自评闭环、错因归类、医学单词收割。本工具不擅长大规模长期跨考点的间隔重复调度——那是 Anki / Obsidian SRS 的活。把毕业的题导成 .md 喂回 Obsidian 永久存档。
