乳癌危险因素 Breast Cancer Risk Factors★★★★★ 本科 ~7题(MCQ1,14,22,28,42;EMQ多处)
核心逻辑=雌激素累积暴露越多、风险越高。凡延长雌激素暴露的生殖因素都是危险因素;哺乳、多产、早绝经是保护因素。
类别危险因素陷阱/保护
生殖(雌激素暴露)早初潮 · 晚绝经 · 未产 · 晚育(>30岁初产)晚初潮/哺乳/多产 = 保护(MCQ1,42 反选)
遗传家族史(尤其乳/卵巢癌)· BRCA1/2 突变家族史是主要危险因素(MCQ14-B)
既往病理非典型导管增生 ADH · 小叶原位癌 LCIS · 对侧乳癌史纤维腺瘤个人史不增风险(MCQ28-D)
其他肥胖 · 高龄 · HRT · 电离辐射—
口诀 「早-晚-未-晚」:早初潮、晚绝经、未产、晚育——四个都增雌激素暴露。BRCA / ADH / LCIS 是三大高危既往因素。
🏥 香港乳癌发病年轻化,家族史 + BRCA 者应考虑基因检测与加强监测;ADH/LCIS 在活检发现后需密切随访或化学预防(他莫昔芬)。
⚠️ 反向题(EXCEPT/NOT)必考:晚初潮、哺乳史、纤维腺瘤史常被混入当『危险因素』——它们要么是保护、要么中性(MCQ1,28,42)。
筛查与影像 Screening & Imaging★★★★★ 本科 ~9题(MCQ2,20,26,52,53,59,61,68,69)
钼靶 mammography 是最可靠筛查(≥40 岁、两体位 CC+MLO)。年轻/致密乳腺用超声;可疑微钙化 → 立体定位活检。
情形首选影像/处理要点
筛查钼靶(40岁起、每年/两年)两体位显著更多病灶(MCQ59)
<35岁 / 致密乳腺超声 USG超声查囊肿优于钼靶;任何年龄有用(MCQ47,59)
可疑微钙化(超声阴性)立体定位空心针/真空辅助活检微钙化超声常不显,不能只随访(MCQ26,61)
隐匿原发(腋结+找不到原发)乳腺 MRIMRI 最敏感、找隐匿灶(MCQ68)
钼靶恶性征:簇状微钙化、结构扭曲、毛刺/星状、边界不清;粗钙化 macrocalcification = 良性(MCQ52)。超声恶性征:taller-than-wide(高>宽)、边界不规则、内钙化、血流、皮肤增厚。
🏥 可疑微钙化即使空心针阴性,仍需手术切除/密切随访(取样误差,MCQ69-D);穿刺应在钼靶后做(避免血肿伪影)。
⚠️ ①<35 岁不做超声=错(超声对年轻致密乳腺更有用,MCQ47-C)。②钼靶阴性≠排除(有肿块仍需查/切,MCQ59-E)。③粗钙化是良性、微钙化才可疑(MCQ69-A 反选)。
三联评估 Triple Assessment★★★★ 本科 ~4题(MCQ46,47,57,74)
任何乳块都做三联:①临床(病史+查体)②影像(钼靶/超声)③病理(FNAC 或空心针 core)——三者一致才可靠。
方法特点
FNAC 细针抽吸快、诊室可做;只出细胞学、不能分浸润/原位;无细胞或不确定 → 升级 core
Core / Trucut 空心针出组织学、可分浸润vs原位、测 ER/PR/HER2;FNAC 不确定的下一步
切除活检 Excisional用于确诊需完整切除者(如 ADH、叶状、切净小病灶)
口诀 「临-影-病」三合一。FNAC 不确定 → core(不是 incisional 切开活检)(MCQ74-B 陷阱)。
🏥 FNAC 应『易抽出细胞成分』;若抽出草黄囊液且肿块塌陷 = 单纯囊肿 → 观察(不必送细胞学)。可疑复发/胸壁结节 → core 取组织(MCQ66)。
⚠️ 『年轻不会得乳癌、不需检查』是错误陈述(MCQ74-C);任何持续肿块都要三联。
导管原位癌 DCIS★★★★★ 本科 ~7题(MCQ3,27,35,55,60;EMQ2-Q8)
DCIS = 未突破基底膜的非浸润癌。治疗三原则:①不需腋清(除非疑微浸润/做乳切时选择性前哨)②不需化疗 ③可保乳(切缘足够)。
情形处理
不可触病灶导丝 hookwire / 立体定位引导广切 + 切缘阴性(MCQ27,60)
范围大(>4cm)/ 多灶 / 不能切净单纯乳房切除 simple mastectomy(不需腋清)(MCQ3,35,55)
保乳后辅助放疗 RT;ER+ 可他莫昔芬(选择性,非常规)
腋窝常规不做腋清;仅乳切/疑微浸润时选择性前哨
DCIS 三不:不腋清、不化疗、不一定乳切。切缘标准 ≥2mm。EMQ2-Q8:8mm 中级 DCIS 切缘 >5mm、想保乳 → 乳房放疗(已切净,加 RT)。
🏥 DCIS 多经筛查微钙化发现(线性分支钙化,EMQ3-Q3);立体定位空心针确诊后规划保乳/乳切。
⚠️ ①DCIS 不需 I/II 腋清(MCQ27-B,60-E 反选)。②不需化疗(MCQ35-A)。③『无论大小改良根治』= 过度(MCQ55-C)。
保乳手术 BCS 指征与禁忌★★★★★ 本科 ~7题(MCQ8,12,18,21,25,41,63)
保乳(WLE + 全乳放疗)生存等同乳切、局部复发率略高。核心禁忌 = 不能切净或不能安全放疗。
类别内容
有利/指征T1 小肿瘤、肿瘤/乳房比例合适、单灶、患者意愿
禁忌 Contraindications多灶/多中心、弥漫恶性微钙化、肿瘤过大或乳房过小(比例差)、既往乳腺放疗、孕早中期(RT 禁)、切缘持续阳性
非禁忌(陷阱)老年、淋巴结受累、乳晕下位置、双侧癌本身不是绝对禁忌(MCQ8,21)
禁忌口诀 「多-弥-比-放-孕」:多灶、弥漫钙化、比例差(大瘤小乳)、既往放疗、孕期。大肿瘤 ER+ 想保乳 → 新辅助内分泌/化疗降期(MCQ25)。
🏥 保乳+RT 与乳切生存相当(MCQ12),且不增心理病态(反而降低);孕早期乳癌可手术但 RT 需推迟至产后(MCQ63)。
⚠️ MCQ12:保乳+RT 局部复发率略高于乳切(是 true 陈述,别选『适合所有患者』或『仅绝经前』)。多灶 multifocal 是绝对禁忌(MCQ21-E)。
腋窝处理与前哨淋巴结 Axilla & SLN★★★★ 本科 ~6题(MCQ30,33,45,48,76;MCQ23)
前哨淋巴结 SLN = 引流第一站。SLN 阴性 → 可免腋清;临床阳性(可触/影像证实)结 → 直接 I/II 腋清。
情形腋窝处理
临床腋窝阴性前哨活检 SLNB(蓝染料 + 核素双示踪)
SLN 阴性免腋清(MCQ48-A)
临床阳性/可触结I+II 腋清 axillary dissection(不做前哨)(MCQ33)
预防淋巴水肿皮肤保护 + 轻柔护理;避免患肢重物/感染(MCQ30)
口诀 「哨阴免清、临阳直清」。SLN 可有多个、非只一个;比腋清更少淋巴水肿(MCQ48 反选)。
🏥 腋清并发症:淋巴水肿、血清肿 seroma、伤口感染、胸长神经(翼状肩)/胸背/肋间臂神经损伤——不是桡神经(MCQ76 反选)。
⚠️ 可触/临床阳性结不做前哨、直接腋清(MCQ30-A、33);前哨双示踪(蓝染非唯一,术中冰冻非 100% 可靠)。
乳房切除术式 Mastectomy Types★★★ 本科 ~4题(MCQ6,49,66;EMQ2,5,7)
按切除范围递增:单纯 → 改良根治 → 根治。
术式切除内容适用
单纯 Simple乳腺 + 乳头乳晕 + 胸肌筋膜(保腋淋巴、保胸肌)DCIS、预防性、老年
改良根治 MRM乳腺 + 乳头 + 腋淋巴 I/II + 胸肌筋膜(保胸大肌)浸润癌 + 腋结阳性(MCQ6)
根治 Radical(Halsted)上述 + 切除胸大小肌已基本淘汰
记 「改良根治保胸肌、根治切胸肌」。MRM 切除 EXCEPT = 胸大肌 pectoralis major(保留)(MCQ6)。
🏥 现代趋势为保乳 + 前哨 + 肿瘤整形;乳切后重建与预后无关(MCQ23-D)。
⚠️ MCQ6:MRM 切除项目中胸大肌是保留的(EXCEPT 答案);别与根治术混淆。
纤维腺瘤 Fibroadenoma★★★ 本科 ~4题(MCQ7,57;EMQ2,8,9,10)
年轻女性最常见良性肿块:界清、活动、无痛、<3cm。三联良性 → 观察。
特征内容
人群/表现年轻(20–30岁)、光滑活动、『乳房鼠 breast mouse』
超声低回声、均质、界清、后方回声增强 posterior enhancement
FNAC黏附上皮细胞片 + 裸核背景
处理三联良性 → 观察 + 复查超声;>3cm/增大/患者要求 → 切除
纤维腺瘤超声=界清均质 + 后方增强(区别恶性的后方声影 shadowing)。1cm 三联良性 → 观察(MCQ7,57)。
🏥 与叶状肿瘤在 core biopsy 上可能难分;快速增大或体积大者应切除送病理排除叶状。
⚠️ 超声『后方增强=纤维腺瘤』vs『后方声影=可疑恶性』(EMQ8,10 靠此区分)。
叶状肿瘤 Phyllodes Tumour★★★★ 本科 ~5题(MCQ17,24,32,64,70;EMQ7,8,9,10)
纤维上皮性肿瘤,与纤维腺瘤组织学相关但快速增大、体积大。约 10% 恶性。治疗 = 宽切缘广切 WLE(≥1cm)。
要点内容
表现快速增大的大肿块(常 >5cm)、40–50 岁、界清分叶
恶性率约 10% 恶性;恶性经血行转移肺(非淋巴,MCQ64,70)
腋结淋巴转移少见、无需常规腋清(MCQ17-D)
治疗宽切缘 WLE(≥1cm);大/恶性 → 全乳切除
口诀 「快大叶状、宽切防复」。恶性叶状血行转移肺(不是淋巴,MCQ70-B 陷阱);多中心性不像导管癌(MCQ17-C 反选)。
🏥 45岁 7cm 快速增大乳晕后肿块 = 叶状直到证否(MCQ24);core 提示纤维上皮 + 快速增大 → 叶状(EMQ9-Q3)。
⚠️ MCQ17(NOT true):叶状不像导管癌那样高度多中心;MCQ64:叶状可恶性、可转移肺、非总良性。
乳头溢液与导管内乳头状瘤 Nipple Discharge★★★ 本科 ~4题(MCQ9,19,58,67)
血性 / 单孔溢液最需警惕。最常见良性病因 = 导管内乳头状瘤 intraductal papilloma(也需排除 DCIS/癌)。
要点内容
血性单孔溢液导管内乳头状瘤(MCQ19);导管造影示充盈缺损
检查印片细胞学、导管造影 ductogram、超声、微导管切除;FNA 不是溢液的处理(MCQ9)
处理微导管切除 microdochectomy(单孔病变、诊断+治疗)(MCQ58)
细胞学溢液细胞学阳性率低 low yield(MCQ67-D)
口诀 「血单孔 → 乳头状瘤 → 微导管切除」。单孔血性更可疑病理(非生理,MCQ67-C 反选)。
🏥 双侧多孔乳样/绿色溢液多为生理性/导管扩张;单侧单孔血性需影像 + microdochectomy。
⚠️ FNA 是肿块的检查、不是乳头溢液的处理(MCQ9 EXCEPT);单孔血性不是生理性。
乳痛与 ANDI Mastalgia & Fibrocystic★★★ 本科 ~4题(MCQ10,11,13;EMQ3,6)
周期性乳痛最常见病因 = ANDI(乳腺正常发育与退化异常);良性、随月经波动。
要点内容
周期性乳痛病因ANDI(aberrant normal development & involution)(MCQ10)
治疗一线 γ亚麻酸 gamolenic acid(月见草油);支持性 bra、减咖啡因(MCQ11)
纤维囊性病双侧上外象限消长肿块 + 经前胀痛(EMQ3-Q1)
非周期性/局灶排除肋软骨炎 costochondritis / Tietze、Mondor 病
口诀 「周期痛 = ANDI、首选月见草」。男性有症状乳腺发育也可用他莫昔芬(MCQ13)。
🏥 乳痛绝大多数良性;先三联排除肿块,再对症。局灶固定痛 + 内侧压痛(运动员)想 Tietze 综合征(EMQ6-Q2)。
⚠️ 别把周期性乳痛当肿瘤;但新发进行性硬块(尤其 HRT 史)仍要查癌(EMQ6-Q3)。
男性乳腺发育 Gynaecomastia★★★ 本科 ~5题(MCQ31,50,65,71,73;EMQ7-Q1)
雌/雄激素比例失衡致男性乳腺增生。最常见病理性可识别病因 = 药物;评估必查睾丸。
要点内容
最常病理病因药物 drug-induced(MCQ31);生理性见青春期/老年
致病药物螺内酯、西咪替丁、钙通道阻滞剂、合成类固醇、洋地黄、酮康唑;苯丙胺不致(MCQ50 反选)
病理病因肝硬化、性腺功能减退、睾丸/肾上腺肿瘤、甲亢
评估/处理查睾丸 examine testes、查激素;有症状 → 停药/他莫昔芬(MCQ71,73)
药物口诀 「螺-西-钙-类-地」(螺内酯、西咪替丁、CCB、合成类固醇、洋地黄)。双侧乳晕后块 = 男乳发育(MCQ65)。
🏥 双侧无痛乳大先想生理/药物性,查睾丸排除睾丸瘤(hCG↑);无需常规 MRI/切开活检(MCQ73)。
⚠️ MCQ50:苯丙胺 amphetamine 不致男乳发育(反选);评估首选查睾丸,不是查泌乳素(MCQ73-D)。
乳腺感染 Breast Infection★★★ 本科 ~4题(MCQ40,44;EMQ4,6,9)
分哺乳期(金葡菌)与非哺乳期(乳晕下脓肿、与吸烟相关)。脓肿 = 波动感 → 引流。
要点内容
哺乳期乳腺炎/脓肿哺乳数周、触痛红肿;脓肿有波动感(MCQ40)
处理抗生素(抗葡菌)+ 超声引导穿刺抽吸或切开引流;可继续哺乳/排空(MCQ44-A 错)
非哺乳期乳晕下脓肿 + 瘘管,与吸烟相关(MCQ44-B)
鉴别红肿乳房必须排除炎性乳癌 inflammatory carcinoma
口诀 「哺乳金葡、非哺乳吸烟瘘」。切开引流非唯一治疗(可穿刺抽吸,MCQ44-C 错)。
🏥 红肿热痛乳房若抗生素无效/无脓,警惕炎性乳癌(MCQ37 鉴别诊断必列)。
⚠️ MCQ44:哺乳不必停、切开引流非唯一、四环素不是首选(孕哺期禁);瘘管在吸烟者更常见(唯一 true)。
预后因素与分子分型 Prognosis & Subtypes★★★★ 本科 ~6题(MCQ23,34,43,62,72,75)
腋淋巴结状态是最重要预后因素。分子分型(ER/PR/HER2)决定辅助治疗。
要点内容
最重要预后因素腋淋巴结状态 axillary node status(MCQ34)
不良预后结阳性、高级别、脉管侵犯、HER2 过表达、BRCA1;乳房重建与预后无关(MCQ23-D)
ER/PR 阳性他莫昔芬(绝经前)/ 芳香化酶抑制剂(绝经后)
HER2 阳性曲妥珠单抗 trastuzumab / Herceptin
三阴性 TNBCER/PR/HER2 均阴;更侵袭、常伴 BRCA;不用他莫昔芬/赫赛汀,靠化疗(MCQ62,75)
口诀 「结状态定预后、受体定用药」。三阴 = BRCA 相关、化疗为主(MCQ62-A,75-B);他莫昔芬 = SERM(乳腺拮抗、子宫内膜激动 → 增子宫内膜癌风险,MCQ43-B)。
🏥 年轻/三阴/家族史应提供 BRCA 基因检测(MCQ75-B);老弱 ER+ → 原发内分泌治疗免手术(MCQ49,72)。
⚠️ ①三阴不能用曲妥珠单抗(HER2 阴)、不用他莫昔芬(ER 阴)(MCQ62-B,75-C/D)。②他莫昔芬增子宫内膜癌风险(MCQ43-C 反选)。
解剖·转移·特殊表现 Anatomy & Spread★★★ 本科 ~6题(MCQ4,5,15,29,36,51,54;EMQ1,3)
血供、淋巴引流、好发部位、转移谱、Paget 病等散点考点集合。
主题要点
血供胸廓内(内乳)穿支 · 胸外侧 · 胸肩峰 · 肋间动脉;胸背动脉供背阔肌非乳房(MCQ51 反选)
淋巴引流腋(主)· 内乳 · 锁骨上 · 对侧;滑车上淋巴结不引流乳房(MCQ4 反选)
好发部位外上象限 upper outer quadrant最多(MCQ36)

危险因素=雌激素累积暴露凡延长雌激素暴露的生殖因素(早初潮、晚绝经、未产、晚育)都增风险;哺乳、多产、早绝经、晚初潮是保护/中性。反向题常把保护因素混入。
任何乳块走三联评估临床 + 影像 + 病理三者一致才可靠。FNAC 只出细胞学,不确定就升级空心针 core(不是切开活检);core 可测 ER/PR/HER2。
影像按年龄分层<35 岁/致密乳腺 → 超声;≥40 岁 → 钼靶(两体位)。可疑微钙化 → 立体定位活检(超声常不显);隐匿原发 → MRI。
DCIS 三不原则不需腋清、不需化疗、可保乳(切缘足够)。大/多灶/不能切净 → 单纯乳切(仍不需腋清)。
保乳禁忌记『多弥比放孕』多灶、弥漫恶性钙化、比例差(大瘤小乳)、既往放疗、孕早中期。老年/淋巴结/乳晕下位置本身不是禁忌。
腋窝『哨阴免清、临阳直清』临床腋阴 → 前哨;前哨阴性免腋清;可触/临床阳性结 → 直接 I/II 腋清(不做前哨)。
老弱 ER+ → 原发内分泌治疗高龄虚弱、合并症多、ER/PR 阳性、不耐手术 → 他莫昔芬/芳香化酶抑制剂(原发内分泌治疗),免手术。
快大想叶状、血单孔想乳头状瘤快速增大的大肿块 → 叶状肿瘤(宽切缘 WLE);单孔血性溢液 → 导管内乳头状瘤(microdochectomy)。
红肿乳房别漏炎性乳癌哺乳期红肿多为乳腺炎/脓肿,但抗生素无效/非哺乳/无脓时必须排除炎性乳癌;鉴别诊断题记得列它。
超声『增强=良、声影=恶』纤维腺瘤:界清均质 + 后方增强;恶性:边界不规则 + 后方声影 + taller-than-wide。EMQ 靠这两点区分纤维腺瘤 vs 癌。
