脊髓损伤平面定位 Spinal Cord Level Localization★★★★★ 本科最高频 ~15题(MCQ1,9,19,20,33,36,47,73,85,95,97;EMQ6)
定位三步法:①感觉平面 → ②脊髓节段 → ③对应椎体。牢记皮节标志:乳头 nipple = T4 · 剑突 xiphoid = T7 · 脐 umbilicus = T10 · 腹股沟 = L1。关键陷阱:成人脊髓约在 L1–L2 椎体终止(圆锥 conus),故椎体节段 ≠ 脊髓节段,越往下差得越多。急性期呈脊髓休克 spinal shock:损伤平面以下弛缓性瘫 + 反射消失 + 无病理征——别因反射消失误判为下运动神经元病。
感觉平面脊髓节段临床线索典型题
乳头 nippleT4上肢正常、下肢瘫 → 上/中胸段MCQ9,20,47
剑突T7肋缘水平MCQ47(costal margin→T7)
脐 umbilicusT10下肢瘫 + 肛门松弛 → 胸腰交界(圆锥)MCQ1,84
腹股沟L1圆锥/马尾交界—
鞍区 saddleS2–4马尾 cauda equina:LMN、鞍区麻木、无病理征MCQ5
口诀 「4-7-10 = 乳-剑-脐」:T4 nipple、T7 xiphoid、T10 umbilicus。下肢瘫 + 肛门松弛 + 脐平面 = 胸腰交界(MCQ84 答案 = thoracolumbar junction)。
🏥 高处坠落 fall from height 是本地脊髓损伤最常机制。急性完全性损伤先按 ATLS 处理,脊柱制动;下肢弛缓 + 反射消失 = 脊髓休克期,数周后转为痉挛性。C7 骨折(MCQ36,73)急性期见无痛性尿潴留 + 神经源性(低)血压,不是高血压。
⚠️ 最大陷阱:把弛缓性瘫误判为病变在腰骶(LMN)。MCQ1 感觉平面脐(T10)、下肢弛缓瘫、反射全失 → 答案仍是下胸段脊髓(急性脊髓休克),不是 L2–3 椎体。定位看感觉平面,不看反射有无。
脊髓综合征 Incomplete Cord Syndromes★★★★ 本科 5题(MCQ79,87,103;MCQ73,36 部分)
不完全脊髓损伤按解剖束定位。考试三大常客:中央脊髓综合征、Brown-Séquard(半切)、自主神经反射异常。
综合征机制/人群特征题
中央脊髓 Central cord老人颈椎过伸伤 hyperextension上肢 > 下肢无力,手为著;下肢相对保留MCQ79
Brown-Séquard 半切穿透/单侧压迫同侧运动 + 本体觉丧失,对侧痛温觉丧失(交叉)MCQ87
前脊髓 Anterior脊髓前动脉痛温 + 运动丧失,本体觉保留—
圆锥 ConusL1–2 椎体对称鞍区麻木、早期膀胱功能障碍、混合征—
马尾 Cauda equinaL2 椎体以下不对称、鞍区麻木、LMN、无病理征、尿潴留MCQ5
自主反射异常 Autonomic dysreflexiaT6 以上损伤慢性期膀胱膨胀 → 阵发性高血压 + (经典心动过缓)、下肢痉挛MCQ103
Brown-Séquard 记 「同运本、对痛温」:同侧 motor + proprioception,对侧 pain + temperature。MCQ87:右侧运动/本体觉丢 + 左侧痛觉丢 = 右侧半切,平面 C6。
🏥 中央脊髓综合征在香港老年颈椎病患者跌倒后常见,多数保守/延迟手术;而不稳定骨折致压迫需急诊减压。
⚠️ MCQ103 自主反射异常经典是高血压 + 心动过缓,但若选项只给『HT + tachycardia + 下肢强直』且对手是 spinal shock(低血压)/恶性高热,仍选 autonomic dysreflexia。
腰椎神经根定位 Lumbar Nerve Root★★★★ 本科 4题(MCQ3,18;EMQ7-Q3)
椎间盘突出压下位神经根(如 L4/5 盘压 L5 根,L5/S1 盘压 S1 根)。定位靠皮节 + 肌力 + 反射三联。
神经根感觉(麻木)肌力(无力)反射
L4小腿内侧 medial股四头/踝背屈(胫前肌)膝反射↓
L5小腿外侧 + 足背 dorsumEHL 拇长伸肌(足拇背伸)、足下垂踝反射正常
S1足外侧 + 足底 sole、小趾腓肠肌(足跖屈)、足外翻踝反射↓
口诀 「L5 抬拇、S1 踮脚」:L5 = EHL 背伸拇趾(踝反射保留);S1 = 跖屈踮脚(踝反射消失)。MCQ3 答案 = 小腿外侧麻木 + EHL 无力 + 踝反射正常 = 典型 L5。
🏥 巨大中央型 L5/S1 突出可压马尾 → 鞍区麻木 + 尿潴留 = 外科急诊 MRI(MCQ18 尿潴留)。
⚠️ 别把 L5 和 S1 反射搞混:L5 突出踝反射正常(踝反射由 S1 掌管)。放射至『第 1–2 趾』= L5(EMQ7-Q3)。
上下肢周围神经损伤 Peripheral Nerve Palsy★★★★★ 本科 ~12题(MCQ2,6,25,38,49,50,77,88,96,98;EMQ2)
周围神经看具体动作 + 感觉区(区别于神经根看皮节)。上肢三大神经 + 腓总神经是必考。
神经运动丧失感觉区经典机制
桡 Radial垂腕、伸腕/伸指(MCP)/伸拇/展拇手背虎口 / snuffbox肱骨干螺旋骨折、Saturday night palsy(肘伸保留)
尺 Ulnar爪形手、骨间肌、小指外展、Froment 征+小指 + 环指尺半肘部(cubitus valgus → tardy ulnar palsy)
正中 Median对掌、拇短展、腕管征桡侧 3½ 指掌面腕管综合征(Phalen/Tinel+)
腓总 Common peroneal足背屈 + 外翻丧失 → 足下垂足背 + 小腿外侧腓骨颈压迫
臂丛 Brachial plexusErb(C5-6 肩外展)/ Klumpke(C8-T1 爪手)上肢(不含面部)产伤、锁骨下动脉伤
分神经记『桡=伸、尺=展/内在肌、正中=对掌』。小指外展 = 尺神经(不是桡神经,MCQ25/50 陷阱);足背屈 = 腓总(MCQ2)。
🏥 掰手腕 arm-wrestling / 肱骨干螺旋骨折(MCQ98)= 桡神经 neuropraxia(Holstein-Lewis 骨折),多神经失用,常自愈观察。桡骨远端骨折后突然不能伸拇 = EPL 拇长伸肌腱在 Lister 结节磨损断裂(MCQ88),不是桡神经麻痹。
⚠️ ①Saturday night palsy(桡神经沟)肘伸保留(三头肌支已发出);前臂外侧感觉是肌皮神经、非桡神经(MCQ77)。②肘部尺神经迟发麻痹由 cubitus valgus 外翻牵拉,不是 varus。
骨筋膜室综合征 Compartment Syndrome★★★★★ 本科 5题(MCQ13,35,76,86,90)
室内压升高 → 灌注障碍 → 缺血挛缩。最早、最敏感征象 = 与损伤不成比例的进行性疼痛 + 被动牵拉肌肉剧痛;脉搏消失是最晚征象(脉搏存在绝不能排除!)。诊断金标准 = 测室压;治疗 = 急诊筋膜切开 fasciotomy。
6P 征象出现顺序要点
Pain 疼痛(与伤不成比例)最早被动牵拉痛最敏感
Paraesthesia 感觉异常早神经缺血
Paresis 肌无力中—
Pallor 苍白 / Poikilothermia中晚—
Pulselessness 无脉最晚有脉≠排除;等无脉已太迟
记 「痛在前、脉在后」。MCQ35/76/86/90 都在考:去石膏后被动活动足诱发剧痛 → 测室压(不是等无脉、不是抬高观察);儿童桡骨远端骨折最怕的也是它。
🏥 胫骨骨折 + 石膏后进行性疼痛 = 拆石膏、测室压、紧急筋膜切开。延误 = Volkmann 缺血挛缩(儿童肱骨髁上骨折经典,MCQ8:远端脉搏消失 + 毛细血管充盈差是最灾难并发症)。
⚠️ 反向题(MCQ13 EXCEPT)把『外周脉搏消失』列为『不能best characterize』——因为无脉是晚期、不敏感,早期脉搏在。
儿童跛行 / 髋痛鉴别 Paediatric Limp★★★★★ 本科 ~10题(MCQ10,54,82,99,102;EMQ1,4,12)
按年龄轴快速定位。铁律:发热 + 不能负重 + 全身不适 = 化脓性关节炎,急诊穿刺,先排除它。
年龄疾病特征处理
0–婴儿DDH 发育性髋脱位Ortolani/Barlow+、换尿布外展受限、臀位/头胎/女<6月 Pavlik 吊带(MCQ10)
任何化脓性关节炎 Septic arthritis急重、高热、不能负重、超声大量积液急诊穿刺 + 引流(Kocher 标准)
3–8岁一过性滑膜炎 Transient synovitisURTI 后、低热、X线正常、自限镇痛观察(MCQ54,82)
4–8岁Perthes 股骨头缺血坏死隐匿、股骨头密度改变 + 扁平限制负重/支具
超重青少年SUFE 股骨头骨骺滑脱髋膝痛、被动屈髋时强制外旋双髋正位+蛙位 X线 → 固定(MCQ99)
口诀 「婴 DDH、幼滑膜、童 Perthes、少 SUFE、随时化脓」。膝痛查髋!股骨/髋病变常放射至膝(MCQ102:膝 X线正常 → 拍髋)。
🏥 SUFE 好发肥胖青少年,是本地儿童骨科高频;强制外旋征(obligatory external rotation)= 经典体征。一过性滑膜炎 vs 化脓性用 Kocher 标准(发热/不能负重/ESR↑/WBC↑)区分。
⚠️ 血检全正常 + 不像脓毒 → 一过性滑膜炎选镇痛观察,别过度用抗生素/MRI(MCQ82 答案 = oral analgesics)。EMQ4-Q3『reactive synovitis』= 一过性滑膜炎同义。
骨与软组织肿瘤 Bone & Soft-tissue Tumours★★★★★ 本科 ~12题(MCQ16,28,30,34,53,57,71,75,100;EMQ1,11,13)
三要素定位:年龄 + 部位(骨骺/干骺/骨干)+ 影像征象。活检铁律:恶性软组织肿瘤先 MRI → 空心针 core/tru-cut 活检,禁切除活检(会污染需大切除)。
肿瘤年龄/部位影像征象题
骨软骨瘤 Osteochondroma青少年 · 干骺端(膝周)带蒂/宽基外生骨疣;可多发遗传性;<1% 恶变软骨肉瘤MCQ28,53;EMQ11-Q1(小→观察)
骨样骨瘤 Osteoid osteoma青少年 · 长骨夜间痛、水杨酸缓解;透亮巢 + 硬化边MCQ34;EMQ1-Q4
单纯骨囊肿 Simple bone cyst儿童 · 近端肱骨中心溶骨 + fallen fragment signEMQ13-Q2
骨肉瘤 Osteosarcoma青少年(也可老年继发)· 干骺端膝周Codman 三角、日光放射 sunburst;转移肺MCQ16,30;EMQ13-Q3
Ewing 肉瘤儿童 · 骨干 diaphysis洋葱皮 onion-skin;小圆蓝细胞 + 发热(似感染)EMQ13-Q1
软组织肉瘤/脂肪肉瘤成人 · 深部 >5cm 渐大MRI;禁切除活检MCQ75,100;EMQ11-Q3(广泛切除)
部位口诀:骺端软骨瘤/骨肉瘤、骨干 Ewing。骨肉瘤治疗 = 新辅助化疗 + 广泛切除(『化疗无用』是错的,MCQ16)。深部 >5cm 软组织肿块 = 肉瘤直到证否 → 增强 MRI(MCQ75,100)。
🏥 老年病理性骨折先想转移瘤(乳癌/前列腺/肺/肾/甲状腺)+ 多发性骨髓瘤。注意:多发性骨髓瘤是溶骨,骨扫描常假阴性(MCQ71 陷阱),应用骨骼 X线/MRI。乳癌骨转移致长骨即将骨折 → 预防性内固定(EMQ11-Q2)。
⚠️ ①骨肉瘤/骨软骨瘤在干骺端 metaphysis,不是骨骺 epiphysis(MCQ28,30 反选)。②Codman 三角/日光放射提示但非绝对特异。③恶性软组织肿瘤切除活检 excisional biopsy 是禁忌,选增强 MRI 或空心针。
膝关节运动损伤 Knee Sports Injuries★★★★ 本科 ~8题(MCQ31,74,93,104;EMQ2,5,14)
按肿胀时间快速分类:数小时内血关节 haemarthrosis = ACL / 髌骨脱位 / 骨折;次日缓起肿 = 半月板。
结构机制特征题
ACL 前交叉韧带扭转/急停即刻血关节、pivot shift+、慢性不稳MCQ104;EMQ5-Q3,14-Q2
半月板 Meniscus扭转 + 负重交锁 locking、打软 give-way、关节线压痛、延迟肿;McMurray+MCQ31,74,93;EMQ2-Q6
桶柄样撕裂 Bucket-handle移位半月板卡入不能完全主动伸膝(机械阻挡)EMQ14-Q1
盘状半月板 Discoid先天(双侧)青少年双膝弹响 clunk(对侧可无痛)EMQ5-Q2
髂胫束 ITB syndrome跑步增量外侧痛性弹响EMQ5-Q1
髌骨脱位 Patellar dislocation外翻+屈曲即刻血肿、恐惧征、怕下蹲EMQ14-Q3
口诀 「快肿想 ACL、交锁想半月板」。MCQ31 反复交锁 + 肿 + 50 岁扭伤 = 半月板 → 关节镜。MCQ74:『instability 不稳』是韧带征,不是半月板典型征(半月板是交锁/打软)。
🏥 急诊抽出 frank blood(血关节)+ 扭转伤 = ACL 断裂(EMQ5-Q3);若抽出脂肪滴 = 关节内骨折。
⚠️ 半月板治疗看年龄/症状:年轻交锁 → 关节镜;老年退变 → 先理疗。别把每个膝痛都送置换。
股骨头缺血坏死 AVN of Femoral Head★★★★ 本科 4题(MCQ29,37,52;EMQ2-Q3,4-Q2,9-Q2)
血供中断 → 头坏死塌陷。病因:类固醇、酒精、创伤(股骨颈骨折/脱位)、特发。MRI 最敏感;早期 X线可完全正常;晚期塌陷 → 继发 OA。
要点内容
香港常见病因酒精 · 类固醇 · 创伤后 · 特发(镰状细胞在 HK 罕见,MCQ29 反选)
最敏感检查MRI(X线早期正常,MCQ52)
剂量相关类固醇剂量与风险显著相关
晚期 X线头塌陷 collapse、关节间隙窄、硬化;不会有股骨颈骨折(MCQ37 反选)
治疗早期保髋(减压);塌陷/晚期 → 关节置换(非『所有塌陷都需手术』,MCQ52)
病因口诀 「酒 · 激素 · 创伤 · 特发」(ASTI)。Ficat IV 期 + 移位股骨颈骨折 = 全髋置换(EMQ9-Q2)。
🏥 SLE / 肾移植 / 大剂量类固醇病人是本地 AVN 主力。双侧受累常见 → 查对侧。
⚠️ MCQ52『所有 X线示头塌陷者都需手术』= 错(需结合症状/年龄);MCQ37 最不可能所见 = 『股骨颈骨折』(AVN 是头坏死,非颈骨折)。
髋部骨折治疗 Hip Fracture Management★★★★ 本科 ~6题(MCQ63,72,105;EMQ9,15-Q2)
先分囊内 intracapsular(股骨颈)vs 囊外 extracapsular(转子间)——决定治疗与并发症。
类型血供/并发症治疗题
股骨颈(囊内)· 年轻AVN + 不愈合风险高急诊闭合/切开复位内固定(保头)MCQ63,EMQ9-Q1
股骨颈(囊内)· 老年移位同上半髋/全髋置换(THR/hemiarthroplasty)EMQ9-Q2
转子间(囊外)血供好、愈合佳DHS / 髓内钉(内固定),可早期负重MCQ105,EMQ9-Q3
口诀 「颈内怕坏死、转子外好愈」。转子间内固定后可负重;上楼『健腿先上 good leg first』(MCQ105:右患 → 左脚先上楼)。
🏥 老年脆性髋骨折:48 小时内手术↓死亡率、早期康复负重(MCQ72);低能量跌倒即骨折 = 骨质疏松,非高能量。
⚠️ 别把转子间当股骨颈处理:转子间首选内固定(不是置换,MCQ63-C 错)。髋『置换 THR』术后才需防脱位(屈髋<90°);转子间内固定后无此限制(MCQ105-A 错)。
肩关节前脱位 Anterior Shoulder Dislocation★★★★ 本科 ~6题(MCQ55,70,78,83,106;MCQ91 冻结肩)
最常见前下脱位。处理 = 镇静下闭合复位。年龄是复发最强预测因子——越年轻越易复发。
要点内容
并发症腋神经损伤 · Bankart(盂唇)· Hill-Sachs(肱骨头)· 大结节骨折 · 肩袖撕裂(老年)· 复发 · 冻结肩
复发预测年龄(最强,MCQ70);18 岁首脱复发率 ~90%(MCQ106)
处理镇静下闭合复位 → 制动(MCQ83)
冻结肩主被动 ROM 均↓、不痛、肩袖力量满 → 理疗(MCQ91)
复发口诀『越年轻越复发』(MCQ70=age、MCQ106=90%)。小结节 lesser tubercle 骨折 = 后脱位征(前脱位是大结节,MCQ78 反选)。
